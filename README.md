# Netflix Clone

A Netflix-inspired front-end website built using only HTML and CSS. This project focuses on recreating the visual design and layout of Netflix with a responsive and modern user interface.

## Features

* Responsive design
* Netflix-style homepage layout
* Navigation bar and banner section
* Movie rows and thumbnails
* Hover effects using CSS

## Technologies Used

* HTML
* CSS

## Project Structure

* index.html - Main structure
* style.css - Styling
* images/ - Assets used



## How to Run

1. Download or clone the repository
2. Open the folder
3. Open index.html in your browser
